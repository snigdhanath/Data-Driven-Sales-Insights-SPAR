# Data-Driven Sales Insights: SPAR Austria


## Tools & Techniques
- MS Excel: Data cleaning, analysis, and visualization  
- PowerPoint / PDF: Presenting key findings and recommendations  
- Data Analysis Techniques:Trend analysis, seasonal analysis, performance evaluation, and actionable insights  


## How to Explore the Project
1. Open the "Data" folder to review the Excel analysis and dataset  
2. Read "Report" for detailed insights and recommendations  
3. Check "Presentation" for a summarized visual overview of key findings  

## Key Highlights
- Consolidated and analyzed weekly sales data using MS Excel to uncover trends and seasonal patterns  
- Identified low-performing categories (e.g., cosmetics) and proposed targeted marketing strategies, dynamic pricing, and inventory optimization  
- Integrated customer feedback and external research to enhance insights and support data-driven business decisions  
- Developed actionable recommendations to improve profitability, operational efficiency, and overall shopping experience
